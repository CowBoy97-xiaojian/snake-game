# 贪吃蛇小游戏 - 需求文档 (v1.0)

## 1. 项目概述

- **项目名称**: Neon Snake - 霓虹贪吃蛇
- **项目类型**: HTML5 网页小游戏
- **核心玩法**: 玩家控制蛇移动，吃食物变长，躲避障碍和自身，避免游戏结束
- **目标用户**: 休闲游戏玩家

## 2. UI/UX 需求

### 2.1 视觉风格 - 极简现代

- **配色方案**:
  - 背景: 深灰 `#1a1a2e` 配浅灰 `#16213e`
  - 蛇身: 渐变色 `#00d9ff` → `#0099cc`
  - 食物: 活力橙 `#ff6b35`
  - 障碍/道具: 紫色 `#7b2cbf`
  - UI文字: 白色 `#ffffff`
  -  accent: 薄荷绿 `#00f5d4`

- **字体**:
  - 标题: "Orbitron" (Google Fonts)
  - 正文: "Rajdhani" (Google Fonts)

- **动画效果**:
  - 蛇移动: 平滑过渡动画
  - 吃到食物: 缩放 + 闪烁效果
  - 游戏结束: 淡出 + 震动效果
  - UI按钮: 悬停发光

### 2.2 布局结构

```
┌─────────────────────────────────────┐
│  [Logo]  SCORE: 0   HIGH: 100  [≡]  │  ← 顶部栏
├─────────────────────────────────────┤
│                                     │
│         ┌───────────────┐          │
│         │               │          │
│         │   游戏区域    │          │
│         │   (Canvas)   │          │
│         │               │          │
│         └───────────────┘          │
│                                     │
├─────────────────────────────────────┤
│   [Start]   [Mode: Classic]         │  ← 底部控制
└─────────────────────────────────────┘
```

- **画布尺寸**: 600 x 600 px (响应式缩放)
- **网格大小**: 20 x 20 px (30格 x 30格)
- **响应式**: 小屏幕自适应

## 3. 功能需求

### 3.1 经典模式 (Core)

- [x] 蛇自动向前移动，速度 150ms/格
- [x] 键盘方向键控制移动方向
- [x] 吃食物长度 +1，得分 +10
- [x] 撞墙 = 游戏结束
- [x] 撞自己 = 游戏结束

### 3.2 冒险模式 (Adventure)

- [x] **随机障碍物**: 场地中随机生成 5-8 个固定障碍
- [x] **加速道具**: 吃到后速度提升 20%，持续 10 秒
- [x] **护盾道具**: 吃到后 5 秒内撞墙不死
- [x] **减速道具**: 吃到后速度降低 20%，持续 8 秒

### 3.3 附加功能

- [x] **暂停功能**: 按 ESC 或点击暂停按钮暂停/继续
- [x] **最高分记录**: localStorage 保存历史最高分
- [x] **音效**:
  - 吃到食物: "滴" 提示音
  - 碰撞/死亡: "砰" 音效
  - 道具获取: "叮" 音效
- [x] **快速重开**: 死亡后按空格键立即重新开始

### 3.4 用户交互

| 操作 | 输入方式 |
|------|----------|
| 移动 | 方向键 ↑↓←→ 或 WASD |
| 暂停 | ESC 或 按钮 |
| 开始 | 点击 Start 按钮 |
| 重开 | 空格键 (游戏结束后) |

## 4. 技术实现

- **技术栈**: 纯 HTML + CSS + JavaScript
- **渲染**: Canvas 2D
- **存储**: localStorage (最高分)
- **音频**: Web Audio API (合成音效)
